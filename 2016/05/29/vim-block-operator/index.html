Vim中两个常用的块操作命令

YXiao's Blog
2016-05-29
Linux

本文介绍Vim中两种常用的块操作命令。

块操作

块操作有一个很典型的使用场景：批量给若干行添加备注。

典型的块操作：
0 hjkl I– [ESC]

意思是：

  • 0 → 到行头
  • → 开始块操作
  • hjkl → 移动光标
  • I– → I是插入，插入“–”
  • 按[ESC]键来为每一行生效

可视化选择

可视化选择也是一个常见的操作，一个常见的使用场景是批量给若干行添加或减少缩进。

v hjkl >或<或=

意思是：

  • v → 切换到visual模式
  • hjkl → 移动光标
  • < 或 > → 左右缩进
  • = → 自动给缩进 （这个功能相当强大，我太喜欢了）

结束语

学习Vim就像学钢琴一样，一旦学会，受益无穷。笔者也没有完全掌握Vim的所有命令，从本人的使用情况来讲，建议你每天都学习1到2个新命令，久而久之，相信你会熟练掌握。
